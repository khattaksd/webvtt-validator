# Security Policy

## Supported Versions

This project supports the latest release on the `main` branch.

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue.

Instead, report it privately by emailing the repository owner/maintainers (or using GitHub Security Advisories if enabled).

Please include:
- A description of the vulnerability and impact
- Steps to reproduce
- Any proof-of-concept code (if applicable)
- Suggested remediation (if you have one)

We will acknowledge receipt and work with you on a fix and coordinated disclosure.
