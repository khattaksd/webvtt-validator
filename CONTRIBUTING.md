# Contributing

## Development

- Install dependencies: `pnpm install`
- Run tests: `pnpm test`
- Run full CI locally: `pnpm run ci`
- Build: `pnpm run build`

## Pull Requests

- Keep changes focused and easy to review.
- Add/update tests when changing behavior.
- Ensure `pnpm run ci` passes before requesting review.

## Code Style

This project aims to keep dependencies minimal and code easy to audit.
